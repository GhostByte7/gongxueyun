# 工学云自动打卡

一键自动完成工学云实习平台的上班/下班签到，打开就能用。

---

## 怎么用

### 1. 启动程序

双击 `工学云打卡.exe`。

### 2. 填写账号密码

在「用户账号」标签页输入工学云的手机号和密码。

### 3. 设置打卡时间和地址

分别在「上班打卡」和「下班打卡」标签页设置时间、地址。

### 4. 保存配置

点击底部「保存配置」按钮。

### 5. 测试登录

在「用户账号」页点击「测试登录」，确认账号密码正确。

### 6. 启动自动打卡

点击底部「定时模式」，程序会每天在指定时间自动签到。窗口最小化放着就行，不要关。

---

## 配置说明

| 配置项 | 说明 | 示例 |
|--------|------|------|
| 手机号 | 工学云登录手机号 | `13800000000` |
| 密码 | 工学云登录密码 | `你的密码` |
| 打卡模式 | `每天` / `法定工作日` / `自定义周几` | `每天` |
| 打卡时间 | 基准打卡时间，格式 HH:MM | `08:50` |
| 随机浮动 | 时间上下浮动多少分钟，避免被检测 | `5`（实际在 08:50~08:55 之间） |
| 打卡地址 | 签到记录里显示的地址 | `北京市海淀区xxx` |
| 邮件通知 | 打卡结果发到邮箱，填写 SMTP 信息后启用 | 可选，不填也能用 |

## 文件说明

```
├── 工学云打卡.exe        ← 双击运行这个
├── models/              ← 验证码识别模型（不要删）
├── config.json          ← 你的配置（程序自动生成）
├── user/                ← 登录缓存（程序自动生成）
└── main.log             ← 运行日志（程序自动生成）
```

## 常见问题

**登录失败？**
检查手机号和密码有没有填错。

**打卡失败？**
看右侧日志窗口，会写原因。

**电脑关机了还能打吗？**
不能，程序得一直开着。最小化窗口就行。

**会不会被检测出来？**
程序模拟了正常设备请求，时间有随机浮动，目前没问题。但任何自动化工具都有风险。
